# geoip - fast geoip country library

[![GoDoc](https://godoc.org/github.com/phuslu/geoip?status.svg)](http://godoc.org/github.com/phuslu/geoip)

### Getting Started

```go
package main

import (
	"github.com/phuslu/geoip"
)

func main() {
	println(string(geoip.CountryByIPStr("1.1.1.1")))
}

// Output: US
```
